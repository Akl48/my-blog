## 为什么选择 VitePress？

VitePress 是一个基于 Vite 和 Vue 3 的静态站点生成器，具有以下优势：

- ⚡️ **快速启动**：基于 Vite，开发体验极佳
- 🎨 **简洁美观**：默认主题现代化且专业
- 📝 **Markdown 优先**：专注于内容创作
- 🔧 **高度可定制**：可以自由定制主题和功能
- 🚀 **性能优秀**：构建速度快，页面加载快

## 博客搭建过程

搭建这个博客主要包括以下步骤：

1. 安装 VitePress
2. 配置网站信息
3. 创建文章
4. 部署到 GitHub Pages
