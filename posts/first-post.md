# 我的第一篇博客

## 前言

这是我使用 VitePress 搭建的个人博客的第一篇文章！也是直接使用copilot进行搭建，完全不需要操作，AI快速就帮你搭建并部署好了（唯一需要动手的只有GitHub的相关配置）

## 为什么选择 VitePress？

copilot给了我这样几个选项
1. GitHub Pages + Jekyll (最推荐)
* 特点：GitHub 原生支持，零成本托管
::: info 优势
  1. 直接在 GitHub 仓库中写 Markdown
  2. 自动构建和部署
  3. 免费的 username.github.io 域名
  4. 丰富的主题和插件
:::
2. Hexo
* 特点：基于 Node.js 的快速博客框架
::: info 优势
1. 生成速度快
2. 主题丰富美观
3. 插件生态完善
4. 支持 GitHub Pages 部署
:::
3. Hugo
* 特点：用 Go 语言编写，构建速度极快
::: info 优势
1. 构建速度最快
2. 简单易用
3. 主题精美
4. 支持多种部署方式
:::
4. VuePress / VitePress
* 特点：基于 Vue.js 的静态网站生成器
::: info 优势
1. 界面现代化
2. 适合技术文档和博客
3. Vue 生态集成好
:::
5. Docusaurus
* 特点：Facebook 开源的文档/博客框架
::: info 优势
1. React 技术栈
2. 功能强大
3. 适合技术博客
:::

在我看来AI工具基本都能帮你完成搭建与部署，之前一直想搞搞前端，所以这块也是最后选择了Vue进行生成。
